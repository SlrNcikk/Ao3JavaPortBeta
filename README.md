So ayun di sya ma run for the normal run routine sa IntelliJ, instead follow the instructions para marun ang code mam/ser.

Open Project in Intellij
Left Corner beside the icon and name ng project (Ao3JavaPortBeta) Click yung 3 lines
Press View -> Tool Windows -> Maven
May tab sa right entitled Maven
Ao3JavaPortBeta -> Plugins -> javafx -> javafx:run
Wait a few seconds tas mag run na rin ng window and application
